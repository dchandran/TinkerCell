Objective

This tutorial will show how to draw polygons and shapes using the PartGraphics program and save it as an XML file that is compatible with TinkerCell.

  1. Creating a simple box
  2. Coloring
  3. Align points
  4. Beziers
  5. Arcs

Creating a simple box

Step 1: Select the "dot" icon situated below the arrow icon in the "select or insert points" group.
Step 2: Click four points on the screen. Four dots will appear. These are the control points.



Step 3: Select the line tool, which is the second button in the "choose shape type" group.
Step 4: Select the four dots, one by one, in a circular fashion. Close the polygon by selecting the starting dot. 
Note: we will make the angle 90-degrees later.


Coloring

Step 5: Scroll down until you see the "fill and outline color" group. In the "fill and outline group"...

The first two buttons are: outline color and fill color, resp. The fill can have no gradient, linear gradient, or radial gradient.

The next button and slider controls the color and transparency of the main color.

The next button and slider controls the color and transparency of the second color, which is only used in gradient fills.



Set the first color to white and second color to orange, as shown below.





main color is white with no transparency

second color is orange with no transparency



Step 6: Use the drop-down arrow on the fill (bucket) icon and select "radial gradient". Click and drag on top of the square.

Align points

Step 7: Click on the arrow in the "select or insert points" group at the top.
Step 8: Hold CTRL key while dragging one of the points. The point should align with one another.

Beziers

Step 9: Use the dot icon again to create four more points as shown below:

Step 10: select the "Bezier" tool, which is the last button in the "choose shape type" group. 
Step 11: Select the the top-left corner of the box, then the two new points on the left side, then the bottom-left corner of the box, and then the top-left once again to complete the polygon. Do the same for the right side of the box. You should get the following shapes:

Arcs

Step 12: Create two new points at the top, as shown below.

Step 13: Select the "arc" button, the first button in the "choose shape type" group. Use the two sliders at the bottom of the "choose shape type" group to adjust the shape of the arc.
Step 14: Click on one of the newly created points (from step 12) and then the second, and then back to the first to close the polygon.
 

Step 15: Use the third slider at the bottom of the "choose shape type" group to adjust the pen size.
Step 16: Set the main color (in the "fill and outline color" group) to gray. 
Step 17: Click on each of the polygons that have been drawn.


Step 18: Color the polygons and align the points.


Step 19: Step the desired height and width of the entire figure using the "set width and height" group at the very top.
Step 20: Select "Save" from the File menu. Enter a name. 

Done! You will see an XML file accompanied by a PNG icon file for this figure. The XML file can be used in TinkerCell as the graphical object for a part.
